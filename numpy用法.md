* numpy中的+和*以及**广播**

    #加法，对应项相加
    W = np.array([[1, 2, 3], [4, 5, 6]])
    X = np.array([[0, 1, 2], [3, 4, 5]])
    W + X
    
    >>> array([[ 1, 3, 5],[ 7, 9, 11]])
    
    ##乘法
    #乘以一个整数
    x = np.array([1, 2, 3])
    x * 10
    
    >>> array([10, 20, 30])
    
    #乘以一个不同维度的数组，进行拓展
    x2 = np.array([1, 2, 3], [3, 4, 5])
    x * x2
    
    >>> array([1, 4, 9], [2, 8, 15])
